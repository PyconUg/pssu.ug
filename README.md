# Python Software Society of Uganda Website

A Django project scaffold with Tailwind CSS for the Python Software Society of Uganda (PSSU).

## About

This website serves as the online presence for PSSU, providing information about Python programming resources, community events, and developer support in Uganda.

The site is built with Django and uses a configuration-driven approach inspired by modern conference websites.

## Key Features

- **Configuration-driven**: Site settings managed in `site.json`
- **Custom design system**: PSSU brand colors and components
- **Modern tooling**: `uv` for Python dependencies, Tailwind CSS for styling
- **Scalable structure**: Clean separation of concerns
